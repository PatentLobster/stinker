# brain-kit
Laravel tinker snippet app.

![screenshot](public/screenshot.png?raw=true "Screenshot")


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve::electron
```

### Compiles and minifies for production
```
yarn build::electron
```

### Lints and fixes files
```
yarn lint
```
